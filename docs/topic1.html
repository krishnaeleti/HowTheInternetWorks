Topic 1: Domain Names and ICANN

Domain names are the name of a website, for example the domain name of Google’s website would be “www.google.com”. You can buy domain names online from various websites, such as GoDaddy, Bluehost and etc. These are the things people can type into their search bar in the browser to go to a specific website. They are linked to something called an IP Address.

ICANN stands for Internet Corporation for Assigned Names and Numbers. While it has “Internet” in the name it is not a company that controls the Internet. This is a non-profit organization that makes sure the DNS (Domain Name System) which is like the Yellow Pages of the internet is in order, and coordinates to make sure it runs smoothly.

URL: Universal Resource Locator Breakdown

We use domain names because they allow us to remember the name of websites, and not have to remember the complicated IP addresses of a website.

In the picture "ICANN Example" we can see that ICANN is not a domain name registry but an entity that oversees them. It allows them to coordinate different domain name and registries so there are no overlaps in the domain names and that there is no confusion with which IP addresses belongs to which Domain name.
