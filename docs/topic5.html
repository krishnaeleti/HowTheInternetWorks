Topic 5: W3C,HTML and CSS

W3C, also called the World Wide Web Consortium, is a community lead by the creator of the WWW, Tim Berners-Lee. This organization creates the standards for the Web, such as specifications or guidelines. Since the Web is decentralized there needs to be a common understand among all networks for them to interconnect with one another, W3C oversees that whole operation.

HTML or Hypertext markup language the language you use to tell a web browser how to look. It is code that is converted into a html file, and that html file is what is used to display a page on a website. Like this page you are viewing right now.

CSS or Cascading Style Sheets, a style sheet language and it is something that describes how the different HTML elements should be displayed i.e. in which order, the top or bottom or page, left or right etc.

W3C Standards Example Source

We need the W3C because it establishes standard for websites, so they all function in a similar fashion and there is no confusion in how to access them. This allows the Internet to be scalable because then there are no conflicts if one group of websites uses a different standard than another.

We use HTML because it is a language that helps us create webpages, it also allows us to link to other pages in the WWW.

We use CSS because it allows more customization of webpages and can make unique presentation formats and can be as creative as one wants to be, on any device be it a mobile or computer or even a tablet.
