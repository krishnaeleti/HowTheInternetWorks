Topic 3: DNS

Domain Name System or DNS is used to locate the IP address. We know that to go to google.com we type “google.com” in our web browser. The computer then submits a get request to a DNS server, which will look for the IP address connected to “google.com” and that would be “64.233.191.255” and then the webpage would load on your screen. The main benefit of the DNS is that you don’t have to memorize all those complex numbers of every website you want to visit but rather just the domain name.

We use Domain Name Systems because so that you don’t have to memorize all those complex numbers of every website you want to visit but rather just the domain name.
